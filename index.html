3D MAP OF COVID-19 Follow @sk3rts


This page shows the number of confirmed cases throughout the world. The height of the column represents the number of cases in that specific area. Please be patient while loading and processing the density data (+18,000 points). To view this page you need to have a browser that supports WebGL.
This map is based on open data. The earth map is based on a download from the Nasa Earth observatory. The COVID-19 related information was downloaded from the CSSEGISandData Github Repository.
